# 🛍️ WishU Frontend

감성 기반 큐레이션 서비스 **WishU**의 프론트엔드 레포지토리입니다.  
React 기반으로 구축되었으며, 사용자 위시리스트/상품 등록/챗봇 추천 기능 등을 제공합니다.

---

## 📦 기술 스택

- **React 18** (with TypeScript)
- **React Router**
- **Styled-components** or **CSS Modules**
- **Axios** (API 통신)
- **Create React App (CRA)**

---

## 🚀 설치 및 실행

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm start
